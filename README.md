# Уебсайт на Риолит Инженеринг ЕООД

Модерен уебсайт за строителна компания, готов за хостване на SuperHosting.BG.

## Структура на проекта

```
riolit/
├── index.php              # Начална страница
├── za-nas.php            # Страница "За нас"
├── uslugi.php            # Страница "Услуги"
├── proekti.php           # Страница "Проекти"
├── kontakti.php          # Страница "Контакти"
├── contact-handler.php   # Обработчик на контактна форма
├── includes/
│   ├── header.php        # Общ header
│   └── footer.php        # Общ footer
├── assets/
│   ├── css/
│   │   └── style.css     # Стилове
│   └── js/
│       └── main.js       # JavaScript функционалности
└── README.md             # Документация
```

## Инструкции за хостване на SuperHosting.BG

### 1. Качване на файловете

1. Влезте в cPanel на вашия хостинг акаунт
2. Отворете **File Manager**
3. Навигирайте до папката `public_html` (или `www` в зависимост от конфигурацията)
4. Качете всички файлове от проекта в `public_html`

### 2. Структура на файловете в хостинга

Файловете трябва да са организирани по следния начин:

```
public_html/
├── index.php
├── za-nas.php
├── uslugi.php
├── proekti.php
├── kontakti.php
├── contact-handler.php
├── includes/
│   ├── header.php
│   └── footer.php
└── assets/
    ├── css/
    │   └── style.css
    └── js/
        └── main.js
```

### 3. Настройка на контактната форма

Отворете файла `contact-handler.php` и променете следното:

```php
$to = 'info@riolit-engineering.bg'; // Променете на вашия имейл адрес
```

За да работи изпращането на имейли, трябва да:

1. Създадете имейл акаунт през cPanel (Email Accounts)
2. Използвайте този имейл адрес в `contact-handler.php`
3. Разкоментирайте реда `mail($to, $email_subject, $email_body, $headers);`

### 4. Персонализация

#### Контактна информация

Променете контактната информация в следните файлове:

- `includes/footer.php` - в секцията "Контакти"
- `kontakti.php` - в контактните картички

#### Домейн

Ако имате собствен домейн, той автоматично ще сочи към `public_html` папката.

### 5. Тестване

След качване на файловете:

1. Отворете вашия домейн в браузър
2. Проверете дали всички страници се зареждат правилно
3. Тествайте контактната форма
4. Проверете мобилната версия

## Технически изисквания

- **PHP версия**: 7.0 или по-нова (SuperHosting.BG поддържа PHP 7.x и 8.x)
- **База данни**: Не е необходима (сайтът е статичен с PHP includes)
- **Специални разширения**: Не са необходими

## Персонализация на съдържанието

### Промяна на текстове

Всички текстове са в PHP файловете. Можете да ги редактирате директно:

- `index.php` - начална страница
- `za-nas.php` - информация за компанията
- `uslugi.php` - списък с услуги
- `proekti.php` - информация за проекти
- `kontakti.php` - контактна информация

### Промяна на цветове

Цветовете се дефинират в `assets/css/style.css` в секцията `:root`:

```css
:root {
    --primary-color: #1a5490;      /* Основен цвят */
    --secondary-color: #f39c12;    /* Вторичен цвят */
    /* ... */
}
```

### Добавяне на изображения

Ако искате да добавите изображения:

1. Създайте папка `assets/images/`
2. Качете изображенията там
3. Използвайте ги в HTML кода
